Enterprise PKI Lab using EJBCA

Built by Colleen Cummings

Overview

This project demonstrates a hands-on Public Key Infrastructure (PKI) lab using EJBCA and Docker. It simulates enterprise certificate management workflows, including certificate authority setup, certificate profiles, end entities, and certificate-based authentication.

Key Skills Demonstrated

  • PKI and Certificate Authority administration
  • X.509 certificate lifecycle management
  • EJBCA deployment with Docker
  • End Entity and Certificate Profile configuration
  • PKCS#12 certificate generation and browser import
  • TLS and certificate-based authentication

Project Highlights

  • Deployed EJBCA in Docker with TLS setup
  • Generated and imported SuperAdmin certificate
  • Created Certificate Profile: MyUserCert
  • Created End Entity Profile: MyUsers
  • Created test user and simulated certificate enrollment
